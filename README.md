# Hook Counter
Keep track of hook stages in Dead by Daylight.

![Hook Counter](https://github.com/nemonn/dbd-hook-counter/blob/main/Screenshots/1.png)

## [Download](https://github.com/nemonn/dbd-hook-counter/releases/download/v2.0.1/Hook.Counter.zip)
_Use **Windowed** or **Windowed Fullscreen** in Dead by Daylight._

## How to use
- Add hook stage with `ALT`+`1`/`2`/`3`/`4`
- Reset all hook stages with `ALT`+`0`
- Quit with `ALT`+`ESC`

## Is it safe
**Yes!** This is just an overlay, no different than taking notes on a piece of paper.

## Antivirus
The app is unsigned, so your antivirus might view it as a threat. If that happens, just let it finish scanning until it tells you it's safe.
